# Tailwindcss Blocks

This is a sample preview website built using Tailwind CSS and Tailwind Blocks(https://mertjf.github.io/tailblocks/).

Further, the images are used from https://source.unsplash.com/ of various sizes and tags.

For eg:- https://source.unsplash.com/720x600/?fitness,gym 

This helps in building the website super fast with just the components.
